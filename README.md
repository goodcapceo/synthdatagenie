# Synthetic Data Genie

Generates synthetic transaction data for fraud detection training.

## Tech Stack

- Backend: Python, FastAPI
- Frontend: React, Vite, Tailwind CSS

## Features

- Generates realistic transaction patterns (lognormal amounts, peak hours, geographic coherence)
- 6 anomaly types: impossible travel, rapid transactions, unusual amounts, new categories, odd hours, high-risk merchants
- Filter by region (US cities, Northeast, West Coast, etc.)
- Export as CSV or JSON
- Dashboard with stats and charts
